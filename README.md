# RESTful API for Library
## Author
The author of this app is Jakub Klonowski (jakubpklonowski@gmail.com).

## Requirements
Technologies used include:
- go1.18
- go libraries:
    - github.com/go-sql-driver/mysql v1.7.0
	- github.com/gorilla/mux v1.8.0
    - github.com/rs/cors v1.8.3

## Manual
If you want to host API update config file and run file generated for your OS.

Otherwise you can open project files and run "`go run .`" command

To consume api connect to [localhost:10000/api](localhost:10000/api) and choose subsequent endpoint.
